## Programación Orientada a Objetos

La **Programación Orientada a Objetos** (POO) hace que los _problemas sean más sencillos_ al favorecer una
relación fuerte entre datos y funciones. Esta relación se realiza a partir de estructuras denominadas **_objetos_**.  
Este paradigma se basa en la idea de que cualquier programa está formado por objetos y que todo puede ser representado como tal. 

#### Conceptos básicos de POO:
1. Objeto: elemento del programa que posee sus propios datos y su propio funcionamiento.  
2. Clase: describe un grupo de objetos que contienen una información similar (atributos) y un comportamiento común (métodos).
   * Atributos: variables miembro de la clase.
   * Métodos: funciones miembro de la clase.

#### Características de POO:
* Abstracción
* Encapsulamiento
* Polimorfismo
* Herencia
